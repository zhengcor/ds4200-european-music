Static Heatmap of the Average Audio Features by Country

Instructions: This visualization is interactive. To use it, click on the first set of aggregation radio buttons to select the displayed statistic between mean, median, and mode. The second set of radio buttons select which audio feature you would like to see displayed on the visualization. When hovering over the bars, a tooltip appears with the country the bar represents, the aggregation selected, the numerical value of the bar, and if mode is selected, the count that makes up the mode.
