Interactive Choropleth Map

Instructions: This visualization is interactive. To use it, click on the dropdown to change which audio feature is displayed. Hovering over a country will display an outline on the country and a tooltip with the precise value. The legend displays the range of saturation of color, indicating how each country compares to the others.

Description: The choropleth map displays the average audio features for each of the 10 European countries using geographic positioning. Since this is a map, the graph uses filled country regions as marks to represent the data. For channels, the graph uses color saturation to encode the value of the selected audio feature, where darker shading indicates a higher value. The dropdown allows the user to switch between different audio features. From the map, it can be observed that neighboring countries often share similar color intensities, suggesting regional clustering in musical preferences. For example, Western European countries like France, Belgium, and the Netherlands tend to have similar danceability and valence values. Southern countries like Spain and Portugal stand out for acousticness. This geographic view reinforces the idea that location in Europe influences the audio characteristics of popular music.
